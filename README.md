# seed-plant

# 🌱 Seed Plant Monitoring System

## 📌 Overview

The Seed Plant Monitoring System is a smart application designed to monitor plant conditions such as temperature, humidity, and soil health. It helps in improving plant growth using real-time data.

## 🚀 Features

* Real-time temperature monitoring
* Humidity tracking
* Soil condition analysis
* Interactive dashboard
* Backend API integration
* IoT support (ESP32 compatible)

## 🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js

## 📂 Project Structure

## ⚙️ Installation

### Backend

```
cd backend
npm install
npm start
```

### Frontend

```
cd frontend
npm install
npm run dev
```
